# Global Stable Facts

## Purpose

This file stores short, stable environment or workflow facts that are useful across projects.

## Rules

- Keep only durable facts.
- Prefer links or short references to long documents.
- Move long postmortems to `contexts/<project>/`.

## Entries

- Hermes is used as a context and experience layer; Cursor remains the code execution layer.
- Telegram is an optional entry channel, not the primary goal of the Hermes integration.
