# Global User Profile

## Purpose

This file stores short, stable user preferences that may apply across projects.

## Rules

- Keep entries short.
- Do not store secrets.
- Do not store long incident reports here.
- If a preference conflicts with Cursor Rules or current user instructions, Cursor Rules and current instructions win.

## Entries

- 默认使用中文输出，除非用户明确要求英文。
- 高风险动作必须先说明风险、验证方式和回滚方案。
